.save Files

What is it?

.save files are a solution to one of my biggest problems as a game dev, saving progress. These files are save files for games. I haven't released any games that requrire them but I will eventually.

Why?

Unforuantely, I can't save data normally so I need external save files. I won't go into to much detail about how these files work but to keep these files secure, they are encrypted with multiple ciphers that only the games understand. Although saving is more manual, it has one huge benefit, portability. If you put a save file onto a usb, you can use a completely different computer and still play on your save. This is cheating but because saving progress is manual, you can fail a section and reload the file to try again.
