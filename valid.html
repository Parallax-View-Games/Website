Valid SS Commands

This page contains a list of all valid scratch script commands, conditions and reporters. Please keep in mind that this is a list of all all the commands that the interpreter currently supports. There are more commands that I still need to add to the interpreter so until then, only use these when writing scratch script code.

Movement:

  • /move.left(value)
  • /move.right(value)
  • /move.up(value)
  • /move.down(value)
  • /move.setx(value)
  • /move.sety(value)
  • /move.dir(value)
  • /move.leftturn(value)
  • /move.rightturn(value)
  • /move.setdir(value)
  • /move.lockdir()
  • /move.unlockdir()
  • /move.flipdir()

Sprite/costume:

  • /costume.hide()
  • /costume.show()
  • /costume.setsize(value)
  • /costume.say(text)
  • /costume.set(text)
  • /costume.next()

Control:

  • /control.wait(value)
  • /control.stop()

Special commands:

  • /if.[condition](code)
  • /repeat.[value](code)
  • /define.name(code)
  • /run.name()
  • /var.create(name)
  • /var.set_name(text or value)
  • /var.add_name(value)

Conditions:

  • [key.pressed]
  • [touching.mouse]
  • [touching color.hex color]
  • [value = value]
  • [value > value]
  • [value < value]
  • [string = string]

Reporters:

  • "Variable name"
  • "xpos"
  • "ypos"
  • "dir"
  • "mousey"
  • "mousex"
  • "mousedir"
  • "size"
  • "costume number"
  • "costume name"
