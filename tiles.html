Tiles

What is it?

Tiles is a simple mobile game about placing tiles on a grid to defeat the opponent. There are 4 types of tiles, attack, blank, trap and flip.

Where can i play it?

The apk file can be downloaded from my itch.io page or you can play it in the browswer. Disclaimer: Tiles is intended for mobile so its better to play on a phone but the web version can be played on any device.
