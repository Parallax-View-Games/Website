About me

This page has everything that anyone would need to know about me.

My goal

I just want to make games and drink coffee. I don't have a lot of experience but I figured out a way to make decent games and all of my games so far have been made using this trick.

My long term goal

I am a student studying computer science because I want to become a full time software developer. Right now, I want to share what I've been creating and I want to help young devs get their games out to the public. You have to start somewhere after all.

How I make games

I use multiple apps and websites to build them but my games are made using Turbowarp. Turbowarp is a modded version of the scratch editor which allows me more control and it gives me many features that help me make more complex games. The main reason why I use turbowarp is the packager, this allows me to export the projects as html files. Finally, I use electron and cordova. These programs allow me to turn the html files into apps.

Why I'm doing this

I just want to make games but becuase of my experience, this is the only way I know how. But if the result is good then why does it matter that I'm using scratch. If anything, its more impressive. If someone built a chair using a hammer and nails, you wouldn't be impressed. But, if someone built a chair using a spoon and forks, you would be impressed.

Feedback

I am open to suggestions but if you want me to make a game for you then you have to donate some money to me on itch.io. However, i offer a free service for scratch users. If you send me an sb3 file of your scratch game or a link to it, I will turn that game into an app for you.

My thoughts on AI

I hate it and every aspect of its existence. It is removing all creativity from the internet. It is painful to see how it can completely remove jobs from the tech industry. My goal is to become a software developer but AIs are starting to take over this side of tech and its pretty scary. I will admit that I have used AI to help me with cordova and electron but I would rather quit game dev than use AI assets. Sometimes it feels like its needed but to any young devs out there, don't use AI if you truly want to make games.
